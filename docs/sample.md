## Sample test

Results file contains brief information of the last test run, also the overall status of the test (pass/ fail)

### Example

**Test data file**

```json
[
    {
        "name": "person1",
        "reference_id": "r1"
    },
    {
        "name": "person2",
        "reference_id": "r2"
    },
    {
        "name": "person3",
        "reference_id": "r3"
    }
]
```

**Test case file**

```json
[
    {
        "testId": "test_unknown",
        "testDescription": "a sample test",
        "steps": [
            "insert(person1)", 
            "insert(person2)", 
            "identify(person3, person1, person2)", 
            "delete(person3)"
        ]
    }
]
```
